# Security Policy

## Supported Versions

We actively provide security updates for the following versions of VexCTX:

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

We take the security and privacy of VexCTX very seriously. If you find a security vulnerability, please do **not** report it publicly via a GitHub issue. Instead, report it privately to the maintainers.

Please send security reports to:
* **suryanshsingh5654@gmail.com**

In your report, please include:
1. A description of the vulnerability and its potential impact.
2. Step-by-step instructions (or a proof-of-concept script) to reproduce the issue.
3. Your recommended fix or mitigation, if any.

Once received, we will:
* Acknowledge receipt of your report within 48 hours.
* Work on a patch to resolve the issue as quickly as possible.
* Notify you when the issue has been fixed and coordinate a public release timeline.
